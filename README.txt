MADE IN TEAMWORK WITH FELIX

----EDIT MODE----
Left click a tile to interact with it. Currently hovered tile is red.
If the tile is empty or blocked it switches to the other.
If the tile is a special tile it gets selected and turns yellow,
you can then press the tile you want to move it too.
Space will start the simulation witht the starchaser astronaut in the position of the spaceship.
----SIMULATION MODE----
red tiles are tiles on the closed list
blue are on the open list
green are the path the starchaser will take
if no path is found the application enters edit mode again


starchaser will get tired after walking 60 tiles aka 30 seconds
he will rest for 5 seconds
